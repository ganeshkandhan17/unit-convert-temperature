# Unit Converter in Java

This repository contains a Java program that serves as a comprehensive unit converter for temperature scales. The program can convert between Celsius, Fahrenheit, and Kelvin.

<h3>🌍Features</h3>
<ul>
      <li>Interactive menu for easy navigation.</li>
      <li>Six conversion options:</li>
      <ol>
            <li>Celsius to Fahrenheit</li>
            <li>Fahrenheit to Celsius</li>
            <li>Fahrenheit to Kelvin</li>
            <li>Celsius to Kelvin</li>
            <li>Kelvin to Celsius</li>
            <li>Kelvin to Fahrenheit</li>         
      </ol>
      <li>Error handling for invalid inputs.</li>
      <li>Continuous calculation until the user chooses to exit.
</li>
</ul>

<h3>🛠️ Tech Stack:</h3>
<ul>
  <li>Java</li>
</ul>
